## Setup
To run this project, install it locally using npm:

### Server
```
$ cd ../backend
$ npm install
$ npm run dev
```
Running in http://localhost:5001


### Client
```
$ cd frontend
$ npm install
$ npm run start
```
Runnin in http://localhost:5000
